Konzole

Konzole slouží k následné úpravě vzhledu uživatelských vstupů.

  • Změna značky

Odstranění

Příkaz: del {index}
Použitelné v metodách: všechny

Odstraní konkrétní vstup

Index = pořadové číslo v seznamu vstupů

Změna značky

Příkaz: del {index} {značka}
Použitelné v metodách: Matematická, Náhodný šum

Mění značku daného vstupu

Index = pořadové číslo v seznamu vstupů

Příkaz pro vypsání značek v konzoli: markers

Značky pro bodový vstup, náhodný šum:
  • ponz.: Některé značky při větší velikosti ztrácí svůj tvar
  • . = Malý kruh
  • , = Čtverec
  • o = Velký kruh
  • v = Trojúhelník (špička směřuje dolů)
  • ^ = Trojúhelník (špička směřuje nahoru)
  • < = Trojúhelník (špička směřuje doleva)
  • > = Trojúhelník (špička směřuje doprava)
  • 1 = Tvar podobný písmenu Y
  • 2 = Tvar podobný písmenu Y (orientovaný dolů)
  • 3 = Tvar podobný písmenu Y (orientovaný doprava)
  • 4 = Tvar podobný písmenu Y (orientovaný doleva)
  • 8 = Osmistěn (velmi podobný kruhu)
  • s = Čtverec (se zakulacenými rohy)
  • p = Pentagon
  • P = Tučné plus
  • * = Hvězda
  • h = Hexagon
  • H = Hexagon (ležící)
  • + = Plus
  • x = Písmeno x
  • X = Písmeno x (tučné)
  • D = Diamant (rovnostranný)
  • d = Diamant (úzký)
  • | = Svislá čára
  • _ = Vodorovná čára
Značky pro vstup z funkce:
  • - NEBO solid = Plná čára
  • -- NEBO dashed = Přerušovaná čára
  • -. NEBO dashdot = Čerchovaná čára
  • : NEBO dotted = Tečkovaná čára
  • (None = Bez značky)

Změna barvy

Příkaz: col {index}
Použitelné v metodách: všechny

Mění barvu daného vstupu

Index = pořadové číslo v seznamu vstupů

Po odelsání příkazu se otevře nové okno s volbou barvy

Změna velikosti

Příkaz: size {index} {velikost}
Použitelné v metodách: Matematická, Sloupcová, Náhodný šum

Mění velikost (tlušťku čar/šířku) daného vstupu

Index = pořadové číslo v seznamu vstupů

Velikost = celé číslo/desetinné číslo

Změna stylu grafu

Příkaz: GPstyle {style}
Použitelné v metodách: Matematická

Mění pozadí grafu

Příkaz pro vypsání stylů v konzoli: ShowMeStyles

Změna se projeví po restartu aplikace

Dostupné styly:

  • Solarize_Light2 = Nažloutlý základní graf (ggplot)
  • seaborn-bright = graf bez mřížky
  • classic = bez mřížky, šedé okolí
  • dark_background = černé pozadí, bez mřížky
  • fivethirtyeight = roztažený základní (bez okrajů)
  • ggplot = základní graf
  • seaborn = modernější font, světle modré pozadí grafu
  • seaborn-whitegrid = seaborn, bílé pozadí
  • seaborn-dark = seaborn bez mřížky
  • seaborn-poster = seaborn bez mřížky, bílé pozadí, popisy os velmým písmem

Speciální příkazy koláčového grafování

Příkaz: explode {index} {value}
Použitelné v metodách: Koláč

Vysune hodnotu z grafu

Index = pořadové číslo v seznamu vstupů

Value = celé číslo/desetinné číslo

Příkaz: stAngle {angle}
Použitelné v metodách: Koláč

Otočí graf (změní jeho počáteční úhel)

Angle = celé číslo/desetinné číslo (úhel -> 0-360)
